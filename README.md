# AI Resume Builder

## Project Overview
AI Resume Builder is a web application designed to help users create professional resumes quickly and effortlessly using AI-powered suggestions, customizable templates, and easy PDF export features. It's user-friendly and accessible to anyone looking to enhance their job application materials.

## Installation
To set up the AI Resume Builder locally on your machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   ```

2. **Navigate to the project directory:**
   ```bash
   cd ai-resume-builder
   ```

3. **Open the `index.html` file in your browser:**
   Simply double-click the `index.html` file or open it through your browser's file menu.

## Usage
1. Open the main page of the AI Resume Builder.
2. Click on "Build Resume" to start creating your resume.
3. Fill in your personal information, work experience, education, and skills as prompted.
4. Preview your resume and utilize the AI suggestions for an optimized presentation.
5. Choose your desired template from the available options.
6. Download your completed resume in PDF format by clicking the "Download PDF" button.

## Features
- **AI-Powered Suggestions:** Get smart recommendations for your professional summary, skills, and job descriptions.
- **Customizable Templates:** Select from a variety of professionally designed resume templates.
- **Easy PDF Export:** Seamlessly download your resume in PDF format with a single click.
- **Interactive Preview:** Preview your resume in real time and make edits as necessary before downloading.

## Dependencies
The project includes the following key dependencies:
- **Tailwind CSS** - A utility-first CSS framework for styling.
- **Font Awesome** - For using vector icons.
- **html2pdf.js** - A library to generate PDF files from HTML content.

You may want to include a package.json file if you intend to manage dependencies with npm. For this project, the external dependencies are directly included through CDN links.

## Project Structure
```
AI Resume Builder/
├── index.html         # Main page for resume creation
├── preview.html       # Page for resume preview and downloading
```

- **index.html**: The primary interface for users to input their resume details, featuring navigation, a hero section, and numerous features highlighted.
- **preview.html**: A template to display the resume and offer options to change templates or download the resume as a PDF.
